# isync 同步工具
文件同步备份

监视指定目录下文件的新增，并将新增文件同步到指定目标目录

## 安装
`npm install -g @tinoq/isync`

## 运行
查看帮助

`isync -h`

```
Usage: isync [options]

Options:
  -V, --version      output the version number
  -s, --src <path>   【必要】源路径
  -d, --dest <path>  【必要】目标路径
  -h, --help         output usage information
```
